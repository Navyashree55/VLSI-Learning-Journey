# Transistor

## Definition
- A transistor is a fundamental semiconductor device used to amplify or switch electrical signals and power.

## Importance
- Transistor is a building block of modern electronics.

## Basic Working
- It controls current flow through three terminals:
  - Emitter
  - Base
  - Collector
- Made using semiconductor materials like silicon.

---

## Transistor as a Switch

- **ON State** → Closed circuit → Current flows  
- **OFF State** → Open circuit → No current flows  

---

## Transistor as an Amplifier

- A small input signal (voltage/current) is given  
- Transistor controls a larger current at the output  
- Output signal becomes stronger (amplified)  

---

## Key Insight

- Transistor does NOT create energy  
- It uses an external power supply (VDD) to increase signal strength
